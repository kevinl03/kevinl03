# Images Directory

Place your project images here:

- `arbitrage.png` or `arbitrage.jpg` - Screenshot/demo of Stablecoin Arbitrage project
- `sil-framework.png` or `sil-framework.jpg` - Screenshot/demo of VCU SIL Framework
- `testing-framework.png` or `testing-framework.jpg` - Screenshot/demo of Testing Framework

## Supported Formats
- PNG (recommended for screenshots)
- JPG/JPEG (for photos)
- GIF (for animations)
- SVG (for diagrams)

## How to Add Images

1. Add your image files to this directory
2. Reference them in README.md using: `![Alt text](./images/filename.png)`
3. Commit and push to GitHub
